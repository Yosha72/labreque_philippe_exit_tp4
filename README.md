# labreque_philippe_exit_tp4

Portail interactif avec Keypad
Ce projet Godot met en place un système de portail verrouillé contrôlé par un clavier numérique (Keypad). 
Le joueur doit entrer le bon code pour déverrouiller le portail et accéder à la téléportation.

Fonctionnalités
Affichage d’un Keypad interactif quand le joueur entre dans le portail verrouillé.
Après 3 erreurs consécutives, le jeu bascule vers une scène Game Over.
Téléportation fluide du joueur vers une destination prédéfinie une fois le portail déverrouillé.
Support du clavier physique en plus des boutons à l’écran.

Le Projet a fini par fonctionner et je suis tomber sur des erreurs qui empèche que l'on voie tout les aspect si dessus.
